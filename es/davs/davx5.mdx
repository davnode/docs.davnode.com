Para sincronizar Davnode en Android, recomendamos usar la aplicación **DAVx⁵**. Aunque puedes comprarla en Google Play Store para apoyar a los desarrolladores, también está disponible de forma gratuita a través de **F-Droid**, un catálogo de aplicaciones FOSS (Software Libre y de Código Abierto).

Esta guía cubre cómo instalar F-Droid y luego usarlo para instalar DAVx⁵.

## 1. Instalar F-Droid

Si ya tienes F-Droid instalado, salta al [Paso 2](#2-install-davx5).

1. Abre tu navegador web en tu dispositivo Android (ej. Chrome, Firefox).
2. Ve al sitio web oficial: **[https://f-droid.org/](https://f-droid.org/)**
3. Toca el botón azul **Download F-Droid** (Descargar F-Droid).
4. Una vez que termine de descargarse el archivo `.apk`, toca **Abrir**.
5. **Aviso de seguridad:**
   - Si tu teléfono dice _"Por su seguridad, el teléfono no tiene permitido instalar apps desconocidas de esta fuente,"_ toca **Ajustes** (o Configuración).
   - Activa la opción **Confiar en esta fuente** (o Autorizar descargas de esta fuente).
   - Toca la flecha de **Atrás** para volver a la pantalla de instalación.
6. Toca **Instalar**.
7. Una vez finalizado, toca **Abrir**.

## 2. Instalar DAVx⁵

1. Abre la aplicación **F-Droid**.
2. **Importante:** Espera un momento a que F-Droid actualice sus repositorios. Verás una barra de carga en la parte superior o un indicador giratorio.
   > _Si los repositorios no se actualizan, desliza el dedo hacia abajo en la pantalla para forzar la actualización._
3. Toca el **icono de Búsqueda** (lupa) en la esquina inferior derecha.
4. Escribe `DAVx5` y busca.
5. Toca en **DAVx⁵** (de bitfire web engineering).
6. Toca **Instalar**.
7. Si se te piden permisos de seguridad nuevamente:
   - Toca **Ajustes**.
   - Permite a F-Droid instalar aplicaciones desconocidas.
   - Toca **Atrás** y toca **Instalar** de nuevo.

## 3. Próximos pasos

Ahora que tienes DAVx⁵ instalado, necesitas conectarlo a tu cuenta Davnode.
